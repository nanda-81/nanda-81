# Repository review

The public repository inventory confirms five original projects with the clearest combined SDE and open-source story:

1. `JobSearchingAgent` — strongest full-stack/product signal.
2. `MindCare-ChatBot` — AI integration and responsible product framing.
3. `CALAMITI` — applied deep-learning and reproducibility signal.
4. `Huffman-Coding` — algorithms, data structures, and file I/O.
5. `Farm-AI` — full-stack/ML application to a practical domain.

`Quiz-Game` and `Shopping-List` remain supporting projects. Forked repositories and the `Design` repository should not lead the profile. The assessment repository is intentionally excluded from all public showcase content.

The next quality pass should make claims match evidence. In particular, project READMEs should distinguish implemented features from future scope, avoid unsupported production-scale language, and explain how a maintainer can run or inspect the project.
